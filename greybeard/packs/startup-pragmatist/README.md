# Startup Pragmatist Pack

The pragmatic engineer who understands stage-appropriate complexity — counter-balance to over-engineering.

## Quick Start

```bash
# Review a design decision through a pragmatism lens
cat proposal.md | greybeard analyze --pack startup-pragmatist

# Understand the complexity tradeoffs for your stage
cat proposal.md | greybeard analyze --pack startup-pragmatist --mode mentor
```

## Focus Areas

- **Scale Alignment**: Is the complexity appropriate for your current scale?
- **Reversibility**: Can you undo this decision cheaply if you're wrong?
- **Time-to-Ship**: What's the opportunity cost of this architecture?
- **Simplicity**: Is there a boring solution that works?

## When to Use This Pack

Use this pack when balancing perfectionism with shipping — especially useful in early-stage companies or during rapid scaling where complexity needs to match reality.
