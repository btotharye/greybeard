# Solutions Architect Pack

Solutions Architect perspective focused on fit-for-purpose, entity modeling, and when not to build.

## Quick Start

```bash
# Review an architecture or system design
cat architecture.md | greybeard analyze --pack solutions-architect

# Understand the architectural tradeoffs and patterns
cat architecture.md | greybeard analyze --pack solutions-architect --mode mentor
```

## Focus Areas

- **Problem Fit**: Is this actually a technology problem?
- **Entity Modeling**: Are the ownership boundaries clear?
- **Build vs Buy**: Is there a simpler, existing solution?
- **Integration Complexity**: What's the coupling cost?

## When to Use This Pack

Use this pack when making architecture decisions — choosing between building, buying, or process changes. Great for system design reviews and integration planning.
