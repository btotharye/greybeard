# Mentor Mode Pack

An experienced Staff or Principal engineer explaining Staff-level reasoning — focused on teaching, not just reviewing.

## Quick Start

```bash
# Get a mentorship perspective on a proposal or decision
cat proposal.md | greybeard analyze --pack mentor-mode

# Learn the why, not just the what
cat proposal.md | greybeard analyze --pack mentor-mode --mode mentor
```

## Focus Areas

- **Pattern Recognition**: What pattern is this an example of?
- **Implicit Knowledge**: Making tacit knowledge explicit
- **Growth**: Teaching, not just correcting
- **Reasoning**: Explaining the mental models behind concerns

## When to Use This Pack

Use this pack when you want deeper learning from reviews — to understand the "why" behind concerns, not just the "what". Great for onboarding, mentoring, and building judgment.
