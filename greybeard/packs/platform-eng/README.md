# Platform Engineering Pack

A thinking framework for platform engineering decisions.

See `PLATFORM-ENG-PACK.md` for usage and `PLATFORM-ENG-EXAMPLE.md` for a real-world walkthrough.

## Quick Start

```bash
# Review a proposal or decision doc
cat proposal.md | greybeard analyze --pack packs/platform-eng/platform-eng.yaml

# Use mentorship mode to understand the reasoning
cat proposal.md | greybeard analyze --pack packs/platform-eng/platform-eng.yaml --mode mentor
```
